🧠 What is FormFlow?

FormFlow is a complete quiz and form ecosystem packed into a single HTML file.
It delivers a smooth, app-like experience without frameworks, servers, or dependencies.

💡 Built as a capstone project to demonstrate how far pure HTML + CSS + JS can go.

✨ Core Features
🧩 Smart Form Builder
Dynamic question creation
Multiple input types
Answer key support for auto grading
⚡ Instant Auto-Grading
Real-time evaluation
Handles:
Single choice
Multiple choice
Detects both missing & extra answers
📊 Analytics Engine
Interactive charts (Chart.js)
Score breakdown
Pass rate insights
Response tracking
